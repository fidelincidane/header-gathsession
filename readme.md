# Header Gathsession

Landing page estatica construida con **HTML + Sass + Vite**, enfocada en practicar maquetacion y estilos CSS.

## Demo en vivo 🌍

👉 **Ver sitio publicado:** [https://fidelincidane.github.io/header-gathsession/](https://fidelincidane.github.io/header-gathsession/)

## Tecnologias 🛠️

- **Frontend:** HTML5 + JavaScript (ES Modules)
- **Estilos:** Sass (SCSS)
- **Build/Dev Server:** Vite
- **Deploy:** gh-pages

## Requisitos 📋

- Node.js 18+
- npm 9+

## Instalacion local 🚀

```bash
# 1) Instalar dependencias
npm install

# 2) Ejecutar en desarrollo
npm run dev
```

## Scripts disponibles ⚙️

- `npm run dev` - inicia servidor local con Vite
- `npm run build` - genera build de produccion en `dist/`
- `npm run preview` - previsualiza el build generado
- `npm run deploy` - publica `dist/` en GitHub Pages

## Despliegue en GitHub Pages 🌐

### Archivos de configuracion clave 📁

- `package.json` (scripts de build/deploy)
- `vite.config.js` (configuracion de Vite)

### Pasos ✅

1. **Construir proyecto**
   - `npm run build`
2. **Publicar en GitHub Pages**
   - `npm run deploy`
3. **Verificar en GitHub**
   - Revisa en Settings -> Pages que la rama de Pages este activa

## Caracteristicas de la pagina ✨

- Header principal con navegacion
- Estructura organizada de estilos con Sass
- Proyecto base para practicar CSS en el master fullstack
